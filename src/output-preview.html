Electron 앱 흐름

앱 바깥 화면의 XML 클릭·드롭 존에서 legacy xmeml을 불러오고, 기존 작업을 유지할 때는 UPDATE XML, 새로 시작할 때만 NEW JOB을 선택한다. 이어 VIDEO 존에 같은 시퀀스의 완성본 영상을 불러온다.

레퍼런스와 SHOT 매핑은 EDIT PANEL에서 설정하고, EXPORT H.264로 내보낸다. OUTPUT은 완성된 파일이 있는 폴더를 연다.

영상과 XML의 역할

완성본 영상 시간이 타임라인의 기준 시계가 된다. XML은 컷 구조를 제공하며 Premiere의 effect, mask, 좌우반전, dissolve 화면은 재현하지 않는다.

Focus Timeline

상세 타임라인은 기본 7초 단위로 전환되고, 하단 FULL 바는 전체 영상 위치를 계속 표시한다. XML에서 비활성화한 클립은 PRIMARY 계산에서 제외된다.

Mixed FPS Preview

영상은 XML의 원본 프레임 속도로 재생하고, UI는 CONFIG.uiFps 기준으로 갱신한다. 둘은 프레임 번호가 아니라 동일한 재생 시간으로 연결되므로 영상·UI·오디오 길이는 변하지 않는다.

개발자용 단독 열기

이 HTML을 브라우저에서 직접 여는 방식은 비영구 개발 미리보기일 뿐이다. Job 저장, 안전한 XML 교체와 공식 Export에는 Electron 앱을 사용한다.
