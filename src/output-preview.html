Electron 앱 흐름

앱 바깥 화면의 XML 클릭·드롭 존에서 legacy xmeml을 불러오고, 기존 작업을 유지할 때는 UPDATE XML, 새로 시작할 때만 NEW JOB을 선택한다. 이어 VIDEO 존에 같은 시퀀스의 완성본 영상을 불러온다.

영상과 XML의 역할

완성본 영상 시간이 타임라인의 기준 시계가 된다. XML 길이와 영상 길이가 1프레임 이상 다르면 정지 상태의 영상 왼쪽 위에 DURATION Δ가 표시된다. XML은 컷 구조를 제공하며 Premiere의 effect, mask, 좌우반전, dissolve 화면은 재현하지 않는다.

Focus Timeline

상세 타임라인은 기본 7초 단위로 전환되고, 하단 FULL 바는 전체 영상 위치를 계속 표시한다. XML에서 비활성화한 클립은 PRIMARY 계산에서 제외된다.

Mixed FPS Preview

영상은 XML의 원본 프레임 속도로 재생하고, UI는 CONFIG.uiFps 기준으로 갱신한다. 둘은 프레임 번호가 아니라 동일한 재생 시간으로 연결되므로 영상·UI·오디오 길이는 변하지 않는다.

Electron app flow

Load legacy xmeml from the XML click/drop zone outside this screen. Choose UPDATE XML to keep the current work, or NEW JOB only when starting over. Then load the finished video of the same sequence into the VIDEO zone.

Set references and SHOT mappings in the EDIT PANEL, and export with EXPORT H.264. OUTPUT opens the folder that holds the finished files.

Shortcuts

Space play/pause · R reset · D debug · H help

Roles of the video and the XML

The finished video's time is the timeline's reference clock. When the XML and video lengths differ by one frame or more, DURATION Δ appears at the video's top-left while paused. The XML provides the cut structure; Premiere effects, masks, flips, and dissolves are not reproduced.

Focus Timeline

The detail timeline pages in 7-second steps, and the bottom FULL bar keeps showing the position in the whole video. Clips disabled in the XML are excluded from the PRIMARY calculation.

Mixed FPS Preview

The video plays at the XML's original frame rate while the UI refreshes at CONFIG.uiFps. They are linked by shared playback time, not frame numbers, so video, UI, and audio lengths do not change.

Standalone developer preview

Opening this HTML directly in a browser is only a non-persistent development preview. Use the Electron app for Job saving, safe XML replacement, and official Export.
